# Yards Brewing (yards-brewing)
Yards Brewing Company is an independent craft brewery founded in 1995 and headquartered in Philadelphia, Pennsylvania. Yards operates a brewery and taproom at 500 Spring Garden Street and distributes its classic and seasonal beers across the Mid-Atlantic. The company emphasizes local craftsmanship, community, and Philadelphia identity. This repository captures the APIs, developer tools, and machine-readable API artifacts for Yards Brewing.

## APIs

No public APIs have been documented. Yards Brewing is a private craft brewery with no developer program, OpenAPI specs, SDKs, or public API surface. Customer-facing digital touchpoints rely on third-party platforms (Toast for gift cards, Eventbrite for event ticketing, a beer finder, and an online shop).
